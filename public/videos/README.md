# Video Assets

## Finland Laboratory Video

Place your Finland Education Laboratory video file here as:
- `finland-lab.mp4` - Main video file for the laboratory section background

### Video Requirements:
- Format: MP4 (H.264 codec recommended)
- Resolution: 1920x1080 or higher
- Duration: 30-60 seconds (will loop automatically)
- File size: Optimized for web (under 50MB recommended)

### Video Content Suggestions:
- Laboratory equipment and technology
- Students working with VR/AR devices
- 3D printing and robotics activities
- Modern classroom environments
- Finnish education methodology in action

The video will be used as a background with a dark overlay for text readability.
